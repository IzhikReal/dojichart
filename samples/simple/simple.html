Simple Usage

Use the controls to manipulate the chart. Select an instrument and set the granularity using the select and buttons above the chart. Change the interval width, candle size and scroll position using the buttons below.

The chart consists of 3 panels, where each panel is rendered within a separate container DIV element. The first is the time labels, the second is the price chart and the third displays the volume histogram. The separation of panels provides versatility of layout and allows for bespoke UI elements between panels, for example toolbars, legends, labels, panel resize handles or tabbed panels.

Please note, the buttons on this page are not part of DojiChart. They are purely for demonstration.
